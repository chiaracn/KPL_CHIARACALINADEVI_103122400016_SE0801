# Deskripsi Program

Program ini dibuat menggunakan JavaScript dan dijalankan menggunakan Node.js. Tujuan program adalah melakukan pemrosesan terhadap berbagai tipe data yang tersimpan dalam sebuah array, kemudian menampilkan hasil pengolahan data tersebut ke terminal.

Program menerima beberapa jenis data, yaitu:

* String angka
* String biasa
* Number
* Boolean

Setiap tipe data diproses dengan aturan yang berbeda menggunakan fungsi `processData()`.

---

# Source Code

## Fungsi Main

```javascript
function main() {
    const data = [
        "123",
        456,
        "hello",
        78.9,
        true
    ];

    for (let i = 0; i < data.length; i++) {
        const result = processData(data[i]);
        console.log(`Item ${i + 1}: ${data[i]} -> ${result}`);
    }
}
```

Fungsi ini bertugas:

1. Menyimpan data ke dalam array.
2. Melakukan iterasi pada seluruh elemen array.
3. Mengirim data ke fungsi `processData()`.
4. Menampilkan hasil ke terminal.

---

## Fungsi processData

```javascript
function processData(data) {

    if (typeof data === "string") {

        if (!isNaN(data)) {
            return `Number: ${Number(data) * 2}`;
        }

        return `Teks: ${data.toLowerCase()} (panjangnya: ${data.length})`;
    }

    if (typeof data === "number") {
        return `Number: ${data * 2}`;
    }

    if (typeof data === "boolean") {
        return `Boolean: ${data}`;
    }

    return "Tipe data tidak dikenali";
}
```

Fungsi ini bertanggung jawab untuk:

* Mengidentifikasi tipe data.
* Melakukan pemrosesan sesuai tipe data.
* Mengembalikan hasil pemrosesan.

# Analisis Clean Code

Program ini sudah menerapkan beberapa prinsip Clean Code:


Hanya bertanggung jawab menjalankan program.
